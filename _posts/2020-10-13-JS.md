##  函数 function

### 声明函数 和 匿名函数

    声明函数：只要声明，在任何地方都可以使用。

    function  函数名(参数){
        函数体
    }

    //调用函数

    函数名();

    匿名函数：如果在使用之前声明，是不可以被调用的。

     var 函数名 = function(参数){
         函数体
    }

    //调用函数

    函数名();


    在JS中，可以把函数名称看作是一个变量名称。



### 函数的参数


    形式参数
    实际参数

    function 函数名（形式参数）{

    }

    函数名（实际参数）

    JS在很久之前，不支持默认值，但是新的标准中添加了，

    arguments  可以获取所有参数
    ...arg     传几个有几个，数组形式


### 回调函数

    如果直接输出函数名称，那么输出的是整个函数。

    回调函数就是将整个函数当作参数使用的方法。


## 返回值

    函数的执行结果就是返回值

    没有返回值的函数叫做执行函数

    有返回值的函数叫做结果函数

    函数的返回值会立即停止函数执行
