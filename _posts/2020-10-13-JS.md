##  函数 function

### 声明函数 和 匿名函数

    声明函数：只要声明，在任何地方都可以使用。

    function  函数名(参数){
        函数体
    }

    //调用函数

    函数名();

    匿名函数：如果在使用之前声明，是不可以被调用的。

     var 函数名 = function(参数){
         函数体
    }

    //调用函数

    函数名();


    在JS中，可以把函数名称看作是一个变量名称。



### 函数的参数


    形式参数
    实际参数

    function 函数名（形式参数）{

    }

    函数名（实际参数）

    JS在很久之前，不支持默认值，但是新的标准中添加了，

    arguments  可以获取所有参数
    ...arg     传几个有几个，数组形式


### 回调函数

    如果直接输出函数名称，那么输出的是整个函数。

    回调函数就是将整个函数当作参数使用的方法。


## 返回值

    函数的执行结果就是返回值

    没有返回值的函数叫做执行函数

    有返回值的函数叫做结果函数

    函数的返回值会立即停止函数执行

## 递归函数

    自己调用自己的函数

    代码执行还是从上到下执行

###  数组 Array

1. 定义数组

    var 变量 = new Array();

    var  变量 = [];

2.给数组添加元素 

    变量[下标] = 数据

3.length属性

    length代表数组长度

4.数组方法

    toString()  把数组转化为字符串

    join() 把数组元素中添加一个指定字符(或空)组合为一个字符串

    pop()  删除数组的最后一个元素,返回的是被删除的元素

        物理删除: 直接真的删除

        逻辑删除:让最后一个元素不要显示(实际上没有删除)

    push()  向数组中添加一个元素,返回的是数组的长度

    shift()  删除数组的首个元素,返回值是被删除的元素

    unshift()  在数组的开头位置添加一个元素,返回值是数组的长度

    splice(位置,删除多少个元素,添加的元素)  操作数组函数

    concat()  数组连接函数,返回值是新的数组,原来的数组不会发生改变

    slice(起始位置,[可无]切割几个) 数组切割函数,返回值是被切割的元素
